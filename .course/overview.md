# Course Overview

## Motors Clinic

A single 90-minute tech club session where students learn to program a BBC Micro:bit to control electric motors. Students wire and program three motor types — brushed DC motors, hobby servos, and stepper motors — using MakeCode block programming.

### Target Students
Grades 6-12, no prerequisites. No prior programming or electronics experience needed.

### Educational Goals
Program a Micro:bit to control three types of electric motors. Understand PWM signals and duty cycles as the foundation for motor control.

### Student & Parent Goals
Robotics interest, STEM exploration, engineering pathway, fun hands-on enrichment.

### Learning Outcomes
1. Wire a brushed DC motor to an H-Bridge and control speed/direction with MakeCode
2. Connect and program a hobby servo to sweep through positions
3. Wire a stepper motor to a ULN2003 driver and program stepping sequences
4. Understand PWM signals and duty cycles as the basis for motor control

### Format
Tech Club — single 90-minute session covering all motor types.

### Scope
1 session, 90 minutes. All three motor types covered in one session. Condensed from an existing 8-module Sphinx course.

### Delivery
Website with sequential lessons. Students receive a physical kit with motor components and follow lesson content on a Hugo site. Programs are built in MakeCode in the browser, loaded from GitHub links in the lessons. No repo, no Codespaces, no external platform.

### Session Flow
Explain → Demonstrate → Practice. Instructor introduces each motor type and its operating principles, demonstrates the circuit, then students build and program it themselves.

### Differentiation
Self-paced progression. Faster students move through more lessons. Extra challenges at some points. Students are not all taught the same thing at the same time.

### Assessment
Working project demonstration. Students show working circuits and programs for each motor type they reach. No formal tests.

### Tier
2 — Website-based lessons, physical kit, browser-based MakeCode programming.

### Standards Alignment
None. Standards candidates reviewed (CSTA 2-CS-02, ISTE 1.5, NGSS MS-ETS1-4) but not formally adopted. This is an informal tech club session.
