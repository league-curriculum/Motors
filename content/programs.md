These are the MakeCode programs used in this course, with links to the GitHub repositories. You can import any of these into the MakeCode editor using **Import → Import URL**.

## Brushed Motors

- [H-Bridge Control](https://github.com/League-Microbit/H-Bridge-Control) — Basic single-speed control of the H-Bridge
- [Duty Cycle](https://github.com/League-Microbit/duty-cycle-buttons) — Controlling the speed of a motor with a programmable duty cycle

## Hobby Servos

- [Servo Sweep](https://github.com/League-Microbit/Servo-Sweep) — Move the servo back and forth

## Steppers

- [Stepper Sweep](https://github.com/League-Microbit/stepper) — Move the stepper back and forth
