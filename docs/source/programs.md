# Programs

These are the programs used in this class, with links to the Github repositories

* Brush Motors
  * [H-Bridge Control](https://github.com/League-Microbit/H-Bridge-Control.git)
    Basic single speed control of the H-Bridge
  * [Duty Cycle](https://github.com/League-Microbit/duty-cycle-buttons)
    Controlling the speed of a Motor with a programmable duty cycle. 
* Hobby Servos
  * [Servo Sweep](https://github.com/League-Microbit/Servo-Sweep.git) Move the servo back and forth
* Stepper
  * [Stepper Sweep](https://github.com/League-Microbit/stepper) Move the stepper back and forth.
